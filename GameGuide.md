# 下载 iopay

LinkStar 游戏是运行在 iotex 公链上的，所以为了运行游戏您得下载一个钱包，以储存自己的资产和跟游戏合约进行互动，下载链接地址：[Iopay Wallet](https://iopay-wallet.iotex.io/)

在网站里你可以选择适合自己的环境，手机钱包或者桌面钱包都可以进行游戏。游戏暂时不支持 web3 钱包，未来将会支持类似 metamask 等流行的钱包。

游玩游戏过程中有已知的几点可能导致无法进行游戏的问题是需要特别注意的：

<font color=#FFC0CB size=3>桌面钱包：</font>

- 请关闭 VPN 的全局模式，以防止无法连接钱包的情况。
- 如果你浏览器有安装 adblock，请暂时关闭 adblock，避免出现连不上服务器的问题。

<font color=#FFC0CB size=3>手机钱包 IOS 版本：</font>

- 新下载的钱包一开始可能并没有发现页面。因为审核的原因，需要自己手动激活 Iopay 的发现页面：进入 safari，输入 https://iotexdefi.com (此处可能需要挂梯子)。下滑，点击“Enable Discover Tab”。登入 ioPay，添加完成！

# 商店界面

游戏开始以后会进入一个稍微长时间的 loading 画面。我们在这里大概需要等 10-20 秒不等。如果实在等待时间太长，建议关闭游戏重新启动再试试。如果实在不行，请使用 VPN 再次尝试。不出意外的话进入游戏之后，游戏会默认打开市场的拍卖页面。或者你可以通过下方四个按钮中的第三个 Market 按钮进入。

![game1](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game1.png)

![game2](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game2.png)

这个界面估计是各位玩家要逗留的时间最多的一个界面了，我们分开几个部分来讲解吧：

## 官方拍卖(Official Auction)

这个是第一次进入游戏默认打开的界面，里面有 20 个 NFT 卡牌等待大家的领取。这个页面里的 NFT 卡牌分成 4 页，每页有6张卡牌，最后一页是2张卡牌。可以在页面下方上下翻页，也可以点击中间页码部分跳转到某一页。

![game3](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game3.png)

点击具体的某张NFT，上面会出现这个 NFT 的详细拍卖信息和品质的信息。如下这个 NFT：

![game4](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game4.png)

背景色是灰色，上面会展示这个 NFT 的总体品质评价，N级卡牌。<font color=#FFC0CB size=3>游戏中有 5 个品质等级,从低到高分别是 N，R，SR，SSR，UR。品质越高，挖矿能力越高，以后在游戏内的表现也会越好。高稀有度的 NFT 非常值得拥有，它可以让你在LinkStar里面玩得更愉快并且获得更高的利益。</font>

总评价取决于 NFT 的八个部位的综合品质。在 NFT 皮肤下面的拥有 8 个水晶球，分别对应这个 NFT 的八个不同部位的品质。部位品质页跟总体品质一样有 5 个等级，分别对应 8 个水晶球的颜色。水晶球的颜色对应关系如下：

- 灰色（N）
- 蓝色（R）
- 紫色（SR）
- 橙色（SSR）
- 彩色（UR）

8 个水晶球的颜色会最终决定 NFT 的总体品质，也就是皮肤的背景。达到一定的分值将会改变 NFT 的背景，变得更加好看。合成的时候我们还会再提到这 8 个水晶球。这里请大家记下来它们的含义。

如果你想查看具体某个部位的样式和品质，可以点开 NFT 右上角的感叹号，点开之后会显示一个悬浮窗口，里面会显示有具体的信息。水晶球下面的是 NFT 的编号，这个编号是游戏唯一的，你可以通过这个 ID 查看某个 NFT 的属性，我们后面会说到。

显示在 NFT 下方的是这个 NFT 的拍卖信息。总共分为 3 个信息：

- 最上层的是这个 NFT 的当前拍卖价格，默认起步价格是 0.01GFT（这个以后可以通过 DAO 调整）。当某个人拍了这个 NFT 以后这个价格会产生变化，最小加价为 0.01GFT。但是如果竞争的人比较多的时候，建议大家出价一个比较有竞争力的价格，不然你的出价被超过以后，虽然 GFT 会返还给你。但是你消耗的 GAS 是不会退的，这个是 iotex 区块链收取的。
- 价格下方显示的地址是当前出价者最高的人的地址，如果跟你的地址不匹配，就说明你没有参与或者参与了但出价被别人赶超了。这个时候你可以重新出价，以抢这个这张卡牌。（当一个 NFT 没有人出价的时候是不会显示这个信息的，就像上图从左到右的第三个）
- 最后一个信息是当前拍卖剩余时间。以 block 的数量计算。（一个 block 是 5 秒）如果有人出价成功，这个时间会重制成 50block。（也就是 250 秒）如果倒计时结束还没有人继续出价的话，这个 NFT 就会被最高出价者获得。玩家成功获得拍卖的卡牌后可以在My Transaction 页签里找到并且领取这个 NFT。

如果你对某张 NFT 感兴趣，可以点击右下方的 bid 按钮。NFT的详情界面显示如下：

![game5](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game5.png)

如刚才所说这里记录了这个 NFT 的所有详情，（后续还会加入其他内容，比如 NFT 的技能）以及它目前拍卖的价格。确认没有问题以后点击 bid。会弹出钱包的确认框，点击确认即可发送交易。这里本来应该等合约执行完毕的，但是为了大家的购买效率我们去掉了强制等待和刷新。待合约执行完毕之后你需要自己切换页签进行刷新。

# 挖矿界面

## NFT 挖矿

挖矿界面可以通过点击主界面下方最后一个按钮 StarMine 来进行跳转。在这里我们可以派遣 NFT 去矿山进行挖矿。你的所得取决于你派遣的所有 NFT 的矿力的总和。派遣的数量没有限制，但是尽量派遣品质高的 NFT，因为它们的挖矿力会很强大。越是品质高，这种优势就越明显。

![game8](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game8.png)

我们可以选择想要派遣 NFT，这个时候它的上方会有一个绿色的勾。然后点击下方派遣按钮，操作钱包确认即可派遣出去。等待区块链操作完毕，手动刷新以后如果成功的话，派遣的 NFT 会罩上一个灰色的遮罩。并且有 dispatched 的文字提示。上方挖矿力和已获取的数值同时也会开始更新。挖矿力是你当前派遣的总挖矿力，已获取是你当前通过 NFT 挖矿获得的 GFT。如果这个时候选择被派遣的 NFT 的话，下方文字会变成召回。可以对已经派遣的 NFT 进行召回。无论是派遣或者是召唤，都会将已经获得的 GFT 的数字清 0，并且直接领取到你的钱包。你也可以点击领取按钮手动领取积攒的 GFT。

## 流动性挖矿

挖矿力上方有一个流动性挖矿，点开可以打开流动性挖矿界面。在这里可以通过质押 Mimo v2 上 GFT-IOTX 的 LP 来挖取 GFS。

<font color=#FFC0CB size=3>关于 GFS：GFS 为总量 3000w 的 GameFantasy 的社区通证。完全 100%公平发放，没有团队份额，没有生态份额。以后会开放更多游戏内的获取方式。可以用于游戏内 GFS 专享的奖金池挖矿，未来预计开放的各种糖浆池的挖矿以及 StarLink 的 DAO 权利。更多通证经济将在游戏上线后公布。</font>

![game9](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game9.png)

第一次打开流动性挖矿需要对流动性挖矿合约进行授权，点击授权按钮并且在钱包确认以后等待 10 秒左右即可打开流动性挖矿开关。这个时候授权按钮会消失，会多出来两个按钮添加和移除。添加是将 LP 质押到合约里面以获取 GFS。移除是将已经质押的 LP 给解除质押。

![game10](https://github.com/GameFantasyDev/StarLinkBetaTest/blob/main/IMG/game10.png)

我们点击添加按钮，将会打开质押界面。这里显示了你当前在 Mimo v2 上做流动性所拥有的 LP 数量（如果是移除则显示的是你已经质押的 LP 数量）。点击 Max 按钮将自动填充你拥有的最大值。点击确认并在钱包操作确认以后将关闭界面并且刷新流动性挖矿界面，你将可以看到自己质押的 LP。同时 GFS 的获取数量也会开始刷新上涨。

# 合成界面

你可以通过下方第二个按钮 Polymerization 来跳转到 NFT 合成界面。这里可以选择 2 个 NFT 进行二合一的合成，以便获取更高品质的 NFT。这也是游戏中唯一可以获得高品质 NFT 的途径。当然合成并非没有没有风险，你的合成可能得到一个不理想的结果。我们的建议如下：

- 两个 NFT 的八个水晶球你想要继承的那个部位的颜色最好是一样的，或者差距不要太大。（例如你想要保留头部的 SSR，那你准备额两个 NFT 头部最好都是 SSR）
- 就算这样做了也不一定能获得最好的结果，但是已经比胡乱来要好得多了。
- 合成是有风险的，如果你觉得你承受不了这个风险的话可以将卡牌挂到自由市场上。如此以来你能获得收入，愿意挑战的人也更有机会获得自己理想的 NFT。
- 高品质的 NFT 的挖矿力和未来的战斗技能都将是更加优秀的，如果可能请尽量获得高品质的 NFT，这将对之后的征途产生很多积极影响。
- 合成是一条漫长艰辛的路，切忌冲动合成。失败不放弃，终有一天你将获得自己理想的伙伴。

如果上面的规则你都了解清楚了，就可以选择两个 NFT 来进行合成了。被选择的 NFT 会在 NFT 的上方用绿色的勾表示，并且放置到上方两个空位上。当两个位置都放置上 NFT 之后，则可以点击合成按钮进行合成。在等待区块链执行完毕后将播放合成动画并且展示你的合成结果。
